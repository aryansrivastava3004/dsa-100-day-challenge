# Day 5 - Find Numbers with Even Number of Digits

## Problem

LeetCode 1295 - Find Numbers with Even Number of Digits

## Approach

Convert each number into a string and use len() to count its digits. Increase the count whenever the number of digits is even.

## What I Learned

- Converting numbers to strings using str()
- Finding the length using len()
- Checking whether a number is even using %
- Counting elements based on a condition

## Difficulty Faced

Realizing that converting a number to a string is a simple and efficient way to count digits.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
