# Day 8 - Missing Number

## Problem

LeetCode 268 - Missing Number

## Approach

Calculate the expected sum of numbers from 0 to n using the mathematical formula n × (n + 1) ÷ 2. Then subtract the actual sum of the array to find the missing number.

## What I Learned

- Using mathematical formulas in programming
- Comparing expected and actual values
- Solving problems with O(1) extra space
- Looking for smarter approaches instead of brute force

## Difficulty Faced

Understanding why the formula n × (n + 1) ÷ 2 gives the sum of all numbers from 0 to n and how it can be used to find the missing value.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
