# Day 6 - Max Consecutive Ones

## Problem

LeetCode 485 - Max Consecutive Ones

## Approach

Maintain a counter for the current sequence of consecutive ones. Reset the counter when a zero is encountered and continuously update the maximum count.

## What I Learned

- Maintaining a running count
- Resetting counters based on conditions
- Tracking maximum values during iteration
- Array traversal techniques

## Difficulty Faced

Understanding when to reset the counter and when to update the maximum count while traversing the array.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
