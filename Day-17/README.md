# Day 17 - Intersection of Two Arrays

## Problem

LeetCode 349 - Intersection of Two Arrays

## Topic

Arrays, Hashing, Sets

## Approach

Convert both arrays into sets to remove duplicate values. Then use the set intersection operator to find the common elements and return the result as a list.

## What I Learned

- Using sets in Python
- Finding the intersection of two collections
- Removing duplicate values automatically
- Writing cleaner and more efficient solutions using built-in set operations

## Difficulty Faced

Understanding that converting arrays into sets automatically removes duplicates and makes finding common elements much simpler.

## Complexity

Time Complexity: O(n + m)

Space Complexity: O(n + m)
