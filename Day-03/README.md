# Day 3 - Richest Customer Wealth

## Problem

LeetCode 1672 - Richest Customer Wealth

## Approach

Traverse each customer's accounts, calculate their total wealth using sum(), and keep track of the maximum wealth encountered.

## What I Learned

- Traversing 2D arrays
- Using sum() on lists
- Finding maximum values while iterating
- Understanding rows and columns in matrices

## Difficulty Faced

Understanding how to process a 2D array and calculate the total wealth for each customer.

## Complexity

Time Complexity: O(m × n)

Space Complexity: O(1)
