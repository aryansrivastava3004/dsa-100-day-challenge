# Day 1 - Running Sum of 1D Array

## Problem

LeetCode 1480 - Running Sum of 1D Array

## Approach

Traverse the array from left to right and update each element by adding the previous running sum.

## What I Learned

- Array traversal
- Running sums
- In-place modification

## Difficulty Faced

Initially, I wasn't sure how to use the previous element to build the running sum.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
