# Day 12 - Majority Element

## Problem

LeetCode 169 - Majority Element

## Topic

Arrays, Hashing

## Approach

Use a dictionary to count the frequency of each element. As soon as an element appears more than n/2 times, return it.

## What I Learned

- Hash Maps (Dictionaries)
- Frequency counting
- Using get() in Python
- Finding the most frequent element efficiently

## Difficulty Faced

Understanding how dictionaries can be used to keep track of frequencies while traversing an array.

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)
