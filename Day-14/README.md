# Day 14 - Two Sum

## Problem

LeetCode 1 - Two Sum

## Topic

Arrays, Hashing

## Approach

Use a dictionary to store numbers along with their indices. For each number, calculate the complement needed to reach the target and check if it has already been seen.

## What I Learned

- Hash Maps (Dictionaries)
- Storing values and indices together
- Finding pairs efficiently
- Optimizing from O(n²) to O(n)

## Difficulty Faced

Understanding how a dictionary can be used to instantly check whether the required complement has already been seen.

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)
