# Day 13 - Best Time to Buy and Sell Stock

## Problem

LeetCode 121 - Best Time to Buy and Sell Stock

## Topic

Arrays, Greedy

## Approach

Keep track of the minimum stock price seen so far. For each day, calculate the profit if the stock were sold on that day and update the maximum profit.

## What I Learned

- Tracking minimum values efficiently
- Calculating profit dynamically
- Greedy problem-solving approach
- Optimizing from brute force to O(n)

## Difficulty Faced

Understanding why keeping track of the minimum price seen so far is enough to calculate the maximum possible profit.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
