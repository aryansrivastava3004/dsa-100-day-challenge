# Day 15 - Valid Anagram

## Problem

LeetCode 242 - Valid Anagram

## Topic

Strings, Hashing

## Approach

Use a dictionary to count the frequency of each character in the first string. Then traverse the second string and reduce the frequencies. If all frequencies match, the strings are anagrams.

## What I Learned

- Hash Maps (Dictionaries)
- Character frequency counting
- Comparing two strings efficiently
- Applying hashing concepts beyond arrays

## Difficulty Faced

Understanding how character frequencies can be used to determine whether two strings contain exactly the same letters.

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)
