# Day 11 - Remove Duplicates from Sorted Array

## Problem

LeetCode 26 - Remove Duplicates from Sorted Array

## Approach

Use the two-pointer technique. One pointer traverses the array while the other keeps track of the position where the next unique element should be placed.

## What I Learned

- Two-pointer technique
- Working with sorted arrays
- In-place array modification
- Efficiently handling duplicate elements

## Difficulty Faced

Understanding why a separate pointer is needed to track the position of the next unique element while traversing the array.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
