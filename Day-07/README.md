# Day 7 - Contains Duplicate

## Problem

LeetCode 217 - Contains Duplicate

## Approach

Use a set to store the elements seen so far. If an element already exists in the set, return True because a duplicate has been found.

## What I Learned

- Introduction to hashing
- Using sets for fast lookups
- Detecting duplicates efficiently
- Difference between lists and sets

## Difficulty Faced

Understanding how sets store unique values and why lookups in sets are more efficient than lists.

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)
