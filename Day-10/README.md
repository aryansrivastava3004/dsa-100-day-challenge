# Day 10 - Move Zeroes

## Problem

LeetCode 283 - Move Zeroes

## Topic

Arrays, Two Pointers

## Approach

Use a pointer to keep track of the position where the next non-zero element should be placed. Traverse the array and swap non-zero elements into their correct positions while maintaining their relative order.

## What I Learned

- Two-pointer technique
- Swapping elements in an array
- In-place modification
- Maintaining the relative order of elements

## Difficulty Faced

Understanding how the index pointer keeps track of the correct position for non-zero elements while traversing the array only once.

## Complexity

Time Complexity: O(n)

Space Complexity: O(1)
