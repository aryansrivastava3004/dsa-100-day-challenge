# Day 4 - Kids With the Greatest Number of Candies

## Problem

LeetCode 1431 - Kids With the Greatest Number of Candies

## Approach

First find the maximum number of candies any kid currently has. Then check if each kid can reach or exceed that value after receiving the extra candies.

## What I Learned

- Using max() to find the largest value
- Working with conditional comparisons
- Creating Boolean result arrays
- Array traversal

## Difficulty Faced

Understanding that we only need to find the maximum value once and compare every kid against it.

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)
